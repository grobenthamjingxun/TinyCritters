# Pangolin Player Dashboard

## Overview

The **Pangolin Player Dashboard** is a web-based dashboard that visualizes pangolin data from a Firebase Realtime Database. It provides a professional, WWF-inspired interface for monitoring each pangolin's **growth stage, happiness, hunger, and last fed time**.

This dashboard is intended for admins or users who want to view and manage pangolin data efficiently.

---

## Features

1. **Realtime Data Display**

   * Displays all pangolins stored in the Firebase Realtime Database under the `pangolins` node.
   * Shows growth stage, happiness, hunger, and last fed time.
   * Updates automatically when the database changes.

2. **Admin Controls**

   * Admins can remove pangolins from the database using the **Remove** button.
   * Configurable via the `isAdmin` flag in the JavaScript file.

3. **Professional Styling**

   * WWF-inspired dark theme with green highlights.
   * Responsive card layout for each pangolin.
   * Hover effects for cards to improve visual feedback.

4. **Uppercase and Readable Layout**

   * All text is in uppercase for better visibility.
   * Each pangolin is displayed in a separate card with clear labels.
